
# AOR 5357-23 Unit 102 v1 — PWA (Build v9: A4 Print Fix)

You should see a badge under the title: **Build v9 — … (A4 print)**.
Print/PDF is now sized for A4 with sensible font sizes, paddings, image limits, and hidden controls.

## Upload steps
1. Upload ALL files to your GitHub repo root (replace existing).
2. Settings → Pages → Deploy from a branch → main / (root) → Save.
3. On your iPad, refresh twice. If still old, delete the app and re-add to Home Screen.

## Notes
- Body base font: 11pt, tables: ~10pt, h1: 16pt, h2: 13pt.
- Photos capped to ~120mm tall on A4.
- Buttons and interactive controls are hidden for print.
